# 賽博燒金紙 Web Prototype：美術素材需求

目前程式已經完成互動雛形，但正式美術請把素材放在 `assets/` 資料夾，並使用下列檔名。程式會自動引用；如果素材不存在，會用簡易 CSS 佔位圖測試互動。

## 必要素材

### 1. `assets/joss-paper-fujin.png`
- 用途：主要可拖曳、燃燒的金紙。
- 建議尺寸：1024 × 1420 px 或同等比例。
- 格式：PNG，建議保留些微透明邊緣。
- 視覺：福金／刈金類型皆可，但建議以「臺灣常見福金」為主。
- 注意：紙面需要有纖維、壓痕、微髒汙、金色反光，不要太乾淨像純色方塊。

### 2. `assets/furnace-mouth.png`
- 用途：畫面上緣的爐口／吸入口。
- 建議尺寸：1200 × 260 px。
- 格式：PNG，透明背景。
- 視覺：不是完整金爐，而是一個俯視或半抽象的「爐口」：黑色狹縫、內部橘金火光、邊緣有金屬或陶土質感。
- 注意：手機上只會露出上方一條，請讓中間火光集中。

### 3. `assets/app-icon.png`
- 用途：首頁 icon 與 PWA icon。
- 建議尺寸：1024 × 1024 px。
- 格式：PNG。
- 視覺：可沿用 App Store 截圖的圓角金紙 icon 語言。

## 建議追加素材

### 4. `assets/bg-texture.jpg`
- 用途：背景材質。
- 建議尺寸：1600 × 2400 px。
- 格式：JPG 或 PNG。
- 視覺：深棕黑、微煙霧、微金粉、非常低調。

### 5. `assets/paper-burn-edge.png`
- 用途：燃燒遮罩邊緣，之後可升級成更真實的燒焦邊。
- 建議尺寸：1024 × 512 px。
- 格式：PNG，透明背景。
- 視覺：不規則焦黑邊緣，搭配橘金火線。

### 6. `assets/ash-fragments.png`
- 用途：紙片碎屑粒子 atlas。
- 建議尺寸：512 × 512 px。
- 格式：PNG，透明背景。
- 視覺：10～20 個不同形狀的小紙片／灰燼碎屑。

## 音效素材，第二階段可補

目前版本先用 Web Audio API 合成音效，不需要檔案。如果要更精緻，建議準備：

### 7. `assets/sfx-paper-rub.mp3`
- 紙張摩擦聲，0.3～0.8 秒，音量小。

### 8. `assets/sfx-ignite.mp3`
- 金紙送入爐口時的短燃聲，0.8～1.5 秒。

### 9. `assets/sfx-complete.mp3`
- 完成音，可是鈴、磬、木魚或輕金屬共鳴，0.8～2 秒。

## 美術方向總結

整體方向不是 emoji、不是卡通貼圖，而是：

臺灣金紙材質 × 深色小金爐 × 橘金火光 × 細緻金粉粒子 × 安靜儀式感

請避免：
- 大紅大紫的廟會風太滿版
- 過度科幻藍紫霓虹
- Q版角色或 emoji 符號
- 過度寫實到像真實燃燒教學
