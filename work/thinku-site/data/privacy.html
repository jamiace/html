Privacy Policy 隱私權政策

Scope / 適用範圍

本政策適用於您使用「LTrust學習平台服務」應用程式期間，我們蒐集、處理、使用您的個人資料之情形。若您透過本應用程式連結至其他網站或服務，其隱私權政策將依各自規範運作，與本政策無涉。

Types and Methods of Data Collection / 蒐集之資料類型與方式

我們可能會蒐集以下類型的資訊：

  • 使用者提供的資訊：Email 地址（登入用）、暱稱或帳號名稱（如有）。
  • 自動收集的資訊：裝置資訊（裝置型號、作業系統版本、裝置 ID）、使用行為（單字測驗紀錄、學習進度、排行榜得分等）、App 效能與錯誤日誌（透過 Crashlytics）。

Purpose of Data Use / 資料的使用目的

  • 提供與維運 App 功能（如：測驗、儲存進度、排行榜）。
  • 優化使用體驗與功能更新。
  • 錯誤追蹤與效能改善。
  • 符合法令規定或配合政府機關調查。

Sharing with Third Parties / 與第三方分享情況

我們不會任意出售、出租、交換您的個人資料給第三人，唯以下情況除外：

  • 經您明確同意。
  • 法律明文規定。
  • 為防止危害生命、身體、自由或財產。
  • 為改善服務或履行契約必要（例如委外維運廠商，並經保密合約約束）。

Third-Party Services Used / 使用之第三方服務

  • Firebase Authentication：使用者登入與帳號管理。
  • Firebase Realtime Database / Firestore：學習進度與測驗資料儲存。
  • Google Analytics for Firebase：使用行為統計。
  • Crashlytics：App 當機與錯誤紀錄追蹤。

請參閱 Firebase 隱私政策：https://firebase.google.com/support/privacy

Data Retention and Deletion Policy / 資料保存與刪除政策

使用者資料將於帳號存續期間內保存。
當使用者申請停用或刪除帳號時，我們將停止該帳號之使用權限。
惟基於法令遵循、帳務紀錄保存、資訊安全或系統管理等必要原因，帳號使用紀錄資料可能仍需於合理期間內保存。
若您希望查詢、更正或申請停止使用您的個人資料，請與我們聯絡。

Children's Data Policy / 兒童資料政策

本應用程式非特別為 13 歲以下兒童設計，若您為監護人並發現孩童曾提供個人資料，請與我們聯繫以進行刪除處理。

Information Security Protection / 資訊安全保護

  • 防火牆與資料加密技術。
  • 權限控管與內部員工保密協議。
  • 第三方服務依其隱私政策運作。

User Rights / 使用者權利

  • 查詢或請求閱覽您的個人資料。
  • 請求補充或更正。
  • 請求刪除。
  • 拒絕進一步處理資料。

請寄信至客服信箱提出申請，我們將於合理期限內處理。

Policy Changes / 政策變更

本政策將依需求隨時更新。若有重大變更，將公告於 App 或官網。
