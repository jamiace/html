## 1. 遊戲概述與勝利條件

《深紅倖存者》是一款以**自動攻擊、移動走位、隨機升級三選一與十五分鐘終局戰**為核心的 Survivor-like 動作遊戲。玩家不必手動按下攻擊鍵；角色會依各武器規則自動尋找目標並施放攻擊，玩家的主要操作是控制位置、規劃武器組合、管理升級選擇，以及在 Boss 戰中避開高密度彈幕。

### 勝利條件

1. 從 00:00 開始生存。
2. 遊戲時間到達 **15:00** 時，終局 Boss「深淵暴君」出現。
3. **擊敗深淵暴君**才算完成一局並進入勝利結算。

### 失敗條件

- 玩家 HP 降到 0，立即結束本局。
- 到達 15:00 並不代表自動勝利；計時器會停留在 00:00，但戰鬥會繼續，直到玩家或 Boss 其中一方倒下。

### 一局的核心節奏

```text
移動躲避 → 自動攻擊 → 擊殺敵人 → 拾取 EXP → 升級三選一
→ 完成最多 6 種武器的配置 → 15:00 Boss 降臨 → 擊敗 Boss
→ 輸入名稱 → 上傳成績 → 查看 Top 50 與自己的名次
```

## 2. 一分鐘快速上手

1. 按下「進入血月廢墟」。
2. 使用 **WASD、方向鍵、滑鼠拖曳或手機虛擬搖桿**移動。
3. 所有攻擊自動施放；不要停在敵群中央。
4. 接近 EXP 晶體後，晶體會被吸向角色。
5. EXP 滿時，從三張卡片中選一張；可按數字鍵 1、2、3。
6. 最多持有 6 種特殊武器，每種最高 Lv.5。
7. 15:00 時 Boss 出現。此後每多花一個完整秒數，總分扣 10 分。
8. 擊敗 Boss 後輸入玩家名稱並送出成績。

> 最重要的觀念：**排行榜不只看通關速度。** 總分同時受到累積 EXP、Boss 超時秒數與結算等級影響。

| 滑鼠移動 | 點擊，或按住滑鼠左鍵拖曳 | 角色朝游標在世界中的位置移動；接近目標點時會自動減速。 |
| 取消滑鼠目標 | 滑鼠右鍵 | 取消目前的滑鼠移動目標。 |
| 暫停／繼續 | Esc | 戰鬥、敵人、武器冷卻與計時一併暫停。 |
| 升級選擇 | 1 / 2 / 3 或點擊卡片 | 選擇對應的升級卡。 |
| 手機／平板 | 左下虛擬搖桿 | 拖曳搖桿控制方向與移動幅度。 |
| 音效 | 開始畫面的音效按鈕 | 切換音效開啟／關閉。 |

### 滑鼠移動細節

- 游標距離角色小於 **9 px** 時，角色視為抵達並停止。
- 距離小於 **55 px** 時會逐漸減速，最低移動倍率為 **0.28×**。
- 一旦按下鍵盤方向或使用觸控搖桿，滑鼠自動移動會被取消。

### 為什麼開局分數是 -100？

玩家從 Lv.1 開始，而分數公式會扣除「目前等級 × 100」，因此在尚未取得任何 EXP 時：

```text
0 EXP − 0 秒超時 − Lv.1 × 100 ＝ -100 分
```

## 5. 戰鬥、受傷與自動攻擊

### 基本角色數值

| 項目 | 初始值 | 說明 |
| :--- | ---: | :--- |
| HP | 50 | 歸零即失敗。 |
| 基礎移動速度 | 158 | 可由「步伐強化」提高。 |
| 碰撞半徑 | 19 | 影響敵人接觸與拾取判定。 |
| 受傷無敵時間 | 0.75 秒 | 每次成功受傷後，短時間不會再次受傷。 |
| 特殊武器欄位 | 6 | 不包含常駐的基礎針彈。 |

### 自動攻擊原則

- 基礎針彈與大部分特殊武器會自動尋找範圍內的敵人。
- 不同武器有不同的索敵距離、施放位置與範圍；站位會直接改變武器效能。
- 當武器施放時找不到合法目標，多數武器不會浪費施放，會等下一個更新時重新嘗試。
- 腐蝕地雷、寒霜脈衝與震地戰鎚不需要目標，會直接在玩家附近生效。

### 受傷來源

- 敵人碰撞傷害。
- 瘟疫噴吐者的遠程彈。
- 膨脹自爆體的爆炸。
- 深淵暴君的扇形彈、連射長槍、環形彈幕與接觸傷害。

### 擊退與控制

- 多數普通敵人可被擊退、暈眩、緩速或凍結。
- 深岩巨像只承受一般擊退量的 10%。
- Boss 不受一般武器擊退，但可受到部分緩速與短暫凍結。
- 全域冰凍道具會讓普通敵人停止；對 Boss 則轉為高幅度緩速。

## 6. 分數與排行榜規則

### 正式分數公式

```text
總分 ＝ 累積取得的 EXP × 1
      − 超過 15:00 的完整秒數 × 10
      − 結算等級 × 100
```

| 構成 | 計算方式 | 意義 |
| :--- | :--- | :--- |
| EXP 分數 | 總 EXP × 1 | 實際拾取並加入累積值的 EXP；雙倍 EXP 期間會按實際取得量計分。 |
| 超時扣分 | 超過 15:00 的完整秒數 × 20 | 15:00.000～15:00.999 尚未形成 1 個完整超時秒；到 15:01.000 才扣 20 分。 |
| 等級扣分 | 結算等級 × 100 | Lv.1 也會扣 100 分。 |

### 分數範例

| 情境 | 總 EXP | 通關時間 | 等級 | 計算 | 總分 |
| :--- | ---: | :---: | ---: | :--- | ---: |
| 剛開局 | 0 | 00:00 | 1 | 0 − 0 − 100 | -100 |
| 準時擊殺 | 7,500 | 15:00.8 | 35 | 7,500 − 0 − 3,500 | 4,000 |
| Boss 戰 25 秒 | 8,000 | 15:25.4 | 40 | 8,000 − 500 − 4,000 | 3,500 |
| 高 EXP 但拖太久 | 10,000 | 16:10.0 | 45 | 10,000 − 1400 − 4,500 | 4,100 |

### 即時分數的特性

- 撿到 1 EXP，分數立即增加 1。
- 升級時，等級加 1，因此分數會立即再扣 100；這是正常現象。
- 15:00 之後，每完成一個額外秒數，分數再扣 20。
- 分數可以是負數。
- Boss 死亡後遊戲立刻結算，因此只有在勝利前已實際拾取的 EXP 會進入本局總 EXP。

### 排行排序順序

1. **分數較高者優先**。
2. 同分時，**總通關時間較短者優先**。
3. 再同分時，**擊殺數較多者優先**。
4. 再同分時，**等級較低者優先**。
5. 仍完全相同時，較早送達伺服器的紀錄優先。

> 排行榜畫面顯示「超出時間」，但同分時的第二順位比較使用完整通關時間。由於所有合法勝利都在 15:00 之後，因此實務上兩者差距一致。

### 伺服器端驗證

- 遊戲會傳送總 EXP、通關時間、等級與擊殺數。
- Google Apps Script 會用相同公式重新計算，排行榜不直接採信瀏覽器傳來的總分。
- 每一局都有唯一 Run ID，避免同一局重複登錄。

## 7. EXP、升級與三選一

### EXP 晶體

- 敵人死亡時在死亡位置留下 EXP 晶體。
- 晶體沒有主動亂飄；平時會回到掉落原點附近。
- 角色進入磁力範圍後，晶體才會被吸向玩家。
- 同一小區域同一幀內的大量晶體可能合併成較大晶體，但總 EXP 價值保留。
- 晶體價值達 5 時使用中型外觀，達 20 時使用大型外觀。

### 升級三選一規則

- 每次升級顯示 **3 個選項**。
- 選項類型包含：武器、永久能力、即時／限時道具。
- 只要仍有合法的武器或永久能力，三選一至少會放入一個永久成長選項。
- 一次取得跨越多級的 EXP 時，會依序連續開啟多次升級畫面，不會漏掉選擇。
- 升級畫面期間戰鬥暫停。

### 武器欄限制

- 最多持有 **6 種特殊武器**。
- 每種武器最高 **Lv.5**。
- 武器欄滿後，不再提供全新武器，但已持有武器仍可繼續升級。
- 基礎針彈為常駐攻擊，不占特殊武器欄。

### 某些道具不出現的條件

| 道具 | 不進入三選一的情況 |
| :--- | :--- |
| 血液補劑 | 目前 HP 已滿。 |
| 再生注射 | 目前 HP 已達最大 HP 的 90% 以上。 |
| 全域磁極 | 地圖上的 EXP 晶體少於 8 個。 |
| 雙倍知識 | 距離 15:00 少於 25 秒。 |

### 基礎針彈隨玩家等級成長

- Lv.1 基礎傷害：**13**。
- 每提升 1 個玩家等級，基礎針彈傷害增加 **2%**。
- 最多計算前 **40 次**成長，即最高增加 **80%**。

| 名稱 | 類型 | 完整效果 |

| 過載核心 | 限時／立即效果 | 12 秒內特殊武器冷卻乘以 0.6，相當於冷卻縮短 40%；常駐軌道武器不受冷卻影響。 |

### 道具效果疊加

- 再次取得同一限時效果時，剩餘時間會被重設為該道具的完整持續時間，而不是在原時間上無限累加。
- 再生、護盾、冰凍、雙倍 EXP、過載與吸血的剩餘時間會顯示於效果列。
- 全域磁極作用於 EXP 晶體，本身不顯示在一般效果列，但會持續吸引。

## 10. 地圖掉落物

除了升級三選一外，場上會定期生成可直接拾取的道具。現行版本的地圖掉落物只有三種：血液補劑、全域磁極、停滯鐘。

| 生成距離 | 通常在玩家周圍約螢幕尺度的 28%～50% 距離生成，另帶隨機偏移。 |

- 從開局出現的大量弱怪，單體威脅低，但數量是前期 EXP 的主要來源。
- 基礎 15 HP、112 速度、6 接觸傷害、1 EXP。
- 後期仍會混入敵群，常成為雷弧鏈、火焰、衛星刃環等範圍武器的連鎖素材。

### 12.2 血獵犬

- 在 01:45 首次登場，首次保證生成 2 隻。
- 具有 0.7～1.15 秒的初始察覺延遲，並以略帶停頓與重新鎖定的方式高速追蹤。
- 高速但血量不高，容易從側面或背後縮短距離。

### 12.3 鐵殼守衛

- 在 02:35 首次登場，首次保證生成 2 隻。
- 150 基礎 HP、移動慢，主要功能是吸收火力並阻擋路線。
- 適合用持續傷害、穿透或集中爆發處理。

### 12.4 瘟疫噴吐者

- 在 03:35 首次登場，首次保證生成 2 隻。
- 約在距離大於 390 時接近，小於 235 時後退，中間距離進行橫移。
- 攻擊距離 520，每 2.4 秒發射一枚基礎傷害 12 的遠程彈。
- 應優先處理，否則多隻噴吐者會形成持續遠程壓力。

### 12.5 膨脹自爆體

- 在 04:50 首次登場，首次保證生成 2 隻。
- 進入玩家 110 距離後開始 0.8 秒充能。
- 充能完成後自爆，爆炸半徑 104，基礎爆炸傷害 22，並隨時間成長。
- 自爆死亡不會掉落 EXP，因為它不是由玩家擊殺。

### 12.6 虛影獵手

- 在 06:35 首次登場，首次保證生成 2 隻。
- 玩家在 520 內時可能啟動突進；先預警 0.55 秒，再以速度 330 突進 0.52 秒。
- 突進冷卻約 3.4 秒。
- 看到紫色預警圈時應改變移動方向，不要沿原路直線逃跑。

### 12.7 深岩巨像

- 在 08:35 首次登場，首次保證生成 1 隻。
- 800 基礎 HP、27 基礎接觸傷害、22 EXP，是高價值但危險的重型目標。
- 只承受 10% 的一般擊退效果；純擊退配置無法長時間阻止其接近。
- 具獨立血條，便於判斷是否值得集中火力。

### 12.8 深淵暴君

- 15:00 出現，7500 HP、35 接觸傷害、60 移動速度。
- Boss HP 不套用一般敵人的時間 HP 成長。
- 擁有四階段彈幕、連射、環形彈幕與召喚能力；詳見第 14 章。

- 清場炸彈對 Boss 單次最多 200 傷害，主要價值常是同時清除召喚怪。
- 重力奇點、火焰、隕石等範圍武器可同時處理 Boss 與召喚怪。
- 超時每秒扣 10 分；在安全與輸出之間做選擇，但長時間拖延通常同時降低分數並增加敵群壓力。

## 15. 線上成績與 Top 50

### 15.1 何時能送出

- 只有擊敗 Boss 的勝利紀錄能送出。
- 失敗局不會出現可用的成績送出流程。
- 玩家名稱最多 **20 個字元**。
- 名稱會保存在瀏覽器本機儲存，下次勝利時自動帶入。

### 15.2 遊戲端排行榜

- 顯示 **Top 50**。
- 欄位依序為：排名、分數、姓名、擊殺數、超出時間、等級。
- 若本局未進入 Top 50，仍會在榜單下方顯示「你的本局成績」與真實名次。
- 送出後遊戲會自動重試讀取排行榜，以等待 Google Sheets 完成寫入與排序。

### 15.3 Google Sheets 保存規則

- 後端依正式排序規則重新排序。
- 試算表只保留排名最高的 **10,000 筆**紀錄；第 10,001 名以後會被刪除。
- Run ID 防止同一局因重複點擊或網路重送而寫入多次。
- 舊版缺少總 EXP 的紀錄無法依新公式重算，因此不會進入新版分數排行。

### 15.4 送出狀態

| 畫面訊息 | 意義 |
| :--- | :--- |
| 正在傳送成績 | 瀏覽器正在向 Apps Script 發送資料。 |
| 正在確認排行榜 | 資料已送出，遊戲正在用 Run ID 尋找本局紀錄。 |
| 成績已登錄，排行榜已更新 | 伺服器已寫入並成功讀回。 |
| 成績已送出；若尚未出現 | 請稍候後按「重新整理」。Google Sheets 偶爾需要短暫處理時間。 |
| 排行榜連線失敗／逾時 | 可能是網路、Apps Script 部署或服務暫時不可用；本局畫面仍可保留並重試。 |


## 16. 配裝思路與實戰策略

以下為依武器機制整理的實戰建議，不是唯一解。隨機三選一可能無法完整取得理想配置，應以當局實際選項調整。

### 16.1 泛用均衡配置

```text
獵殺無人機 + 雷弧鏈 + 寒霜脈衝 + 天墜隕石 + 重力奇點 + 貫星光束
```
- 無人機提供穩定單體輸出。
- 雷鏈處理分散小怪。
- 寒霜脈衝負責解圍與控制。
- 奇點聚怪後，隕石與光束提高總命中效率。

### 16.2 近身防線配置

```text
衛星刃環 + 震地戰鎚 + 寒霜脈衝 + 燼火噴流 + 腐蝕地雷 + 迴旋斬輪
```
- 適合高移速繞圈。
- 刃環持續切割貼身敵人；戰鎚與寒霜脈衝處理包圍。
- 地雷在移動路徑留下防線，斬輪可清理追擊隊伍。

### 16.3 遠端範圍配置

```text
天墜隕石 + 重力奇點 + 貫星光束 + 雷弧鏈 + 爆裂霰彈 + 獵殺無人機
```
- 盡量在敵群外圍移動，讓奇點與隕石清理遠端密集點。
- 霰彈與無人機補足靠近玩家或單體 Boss 的傷害。

### 16.4 Boss 速殺思路

- 優先擁有可持續命中單體的武器：無人機、光束、火焰、近距離霰彈。
- 仍至少保留一種控制或解圍武器，因為 Boss 每失去 5% HP 都會召喚增援。
- 15:00 前保留停滯鐘、護盾或過載效果，可在 Boss 出現後創造輸出窗口。
- 過載核心對有冷卻的特殊武器價值很高，但不會提升衛星刃環本身的常駐頻率。

### 16.5 分數導向策略

- **EXP 不是無條件正收益。**每次升級會額外扣 100 分；前期升級需求低於 100 EXP，跨級瞬間可能讓分數下降。
- 但升級能提高生存與輸出，並讓你在 15:00 後更快擊殺 Boss；單純壓低等級可能使超時扣分更大。
- 雙倍 EXP 同時增加分數與升級速度；應考慮它是否會讓你在 Boss 前跨越多個等級。
- Boss 出現後，每秒扣 10 分。若多花 10 秒去撿 100 EXP，理論上剛好抵消時間扣分，但可能同時升級而再扣 100。
- 同分時擊殺較多者優先，因此在不拖延 Boss 的前提下，清理召喚怪可提供第三排序優勢。
- 同分、同時間、同擊殺時，等級較低者優先。

### 16.6 生存優先順序

1. 先確保移動空間，再追求 EXP。
2. 優先處理瘟疫噴吐者與虛影獵手，降低遠程與突進壓力。
3. 膨脹自爆體開始充能時立刻離開爆炸半徑。
4. 深岩巨像不要只靠擊退，應用持續傷害或集中輸出。
5. 保持地圖掉落物的視覺記憶；需要時再回頭拿護命資源。

## 17. 常見問題

### Q1：15:00 了，為什麼遊戲還沒結束？
因為 15:00 只是 Boss 出現時間。必須擊敗深淵暴君才算勝利。

### Q2：為什麼分數突然下降 100？
你升級了。每個結算等級扣 100 分，因此 Lv.20 比 Lv.19 多扣 100。

### Q3：分數為什麼可以是負數？
分數沒有最低 0 分限制。開局 Lv.1 就先扣 100；EXP 不足、等級高或 Boss 戰拖太久時都可能出現負分。

### Q4：為什麼武器欄滿後看不到新武器？
正常武器上限是 6 種。欄位滿後，只會再出現已持有武器的升級、永久能力與道具。

### Q5：為什麼雙倍知識在快到 15:00 時不出現？
距離 Boss 出現少於 25 秒時，雙倍知識會從三選一候選池移除，避免效果幾乎無法使用。

### Q6：為什麼全域磁極不出現在升級選項？
場上 EXP 晶體少於 8 個時，磁極不進入候選池。

### Q7：Boss 被停滯鐘凍住了嗎？
普通敵人會停止，Boss 則受到大幅緩速，不是完全停止。寒霜脈衝 Lv.5 可以讓 Boss 短暫凍結 0.27 秒。

### Q8：為什麼膨脹自爆體死亡後沒有 EXP？
它若自行完成自爆，仍會增加畫面上的擊殺數，但不掉 EXP；在自爆前由玩家武器擊殺才會正常掉落。

### Q9：為什麼本局沒進 Top 50，畫面仍有我的名次？
排行榜會額外用 Run ID 查詢本局。即使不在 Top 50，仍在榜單下方顯示自己的實際排名。

### Q10：直接雙擊 index.html，為什麼無法載入？
瀏覽器以 `file://` 開啟時，常會阻擋 `game-config.json`。建議透過 GitHub Pages、本機 HTTP 伺服器或遊戲提供的 JSON 選檔功能載入。

### Q11：重新整理後玩家名稱還在嗎？
送出時輸入的名稱會保存在此瀏覽器的 localStorage；清除網站資料、換瀏覽器或換裝置後可能消失。

### Q12：排行榜送出後沒有立刻看到成績？
遊戲會自動重試數次。Google Sheets 寫入與排序可能稍有延遲，可等待後按「重新整理」。
